## Περιγραφή Project
Ένας ταχύτατος και ελαφρύς υπολογιστής που εκτελείται στο τερματικό. Η εργασία εστιάζει στη βασική αλγοριθμική λογική της C++, τη διαχείριση ροής ελέγχου και την αλληλεπίδραση με τον χρήστη.

## Τεχνικά Χαρακτηριστικά & Αρχιτεκτονική
* **Ροή Ελέγχου:** Αξιοποίηση της δομής `switch` για βελτιστοποιημένη επιλογή και εκτέλεση της επιθυμητής πράξης (+, -, *, /).
* **Τύποι Δεδομένων:** Χρήση μεταβλητών τύπου `double` για ακρίβεια σε πράξεις με δεκαδικούς αριθμούς.
* **Έλεγχος Εγκυρότητας (Validation):** * Προστασία έναντι διαίρεσης με το μηδέν.
  * Διαχείριση μη έγκυρων συμβόλων πράξης (μέσω της `default` περίπτωσης στη `switch`).
* **I/O:** Καθαρή αλληλεπίδραση εισόδου/εξόδου μέσω της βιβλιοθήκης `<iostream>`.
